# ChistesAPI

Para empezar a programar hay que

Descargar node.js
con npm descargar express y nodemon con estos comandos en la cmd

npm i -D nodemon 
npm i express

para correr la api y que se actualizen los cambios a la vez que se corre usar en posicion src

npm run dev

para visualizar usar por ahora en web localhost:3000

De buena practica se puede mover toda las carpetas dentro src a otra llamada v1 para asegurar versionamiento y acceder a la misma desde api/v1/chistes
pero como solo va a haber una versión, no se usará.
